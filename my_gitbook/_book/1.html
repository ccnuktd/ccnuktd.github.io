操作系统概述

操作系统概述

初步认识

误区1：没有安装操作系统计算机不能运行程序。

没有安装操作系统计算机仍然可以运行程序，计算机启动CPU会在系统BIOS位置取指令执行，我们还可以看到执行程序而显示的BIOS画面。因此没有安装操作系统的计算机可以运行程序。

操作系统的功能

  • 进程管理
    1. 进程控制（创建，撤销，阻塞，唤醒）
    2. 进程调度（FIFO，短作业优先）
    3. 进程通信
  • 内存管理（现代操作系统不会因为内存过小而导致程序无法运行）
    1. 内存分配
    2. 内存共享
    3. 内存保护
    4. 虚拟内存
  • 设备管理（计算机接上了键盘，鼠标，磁盘，怎么管理）
    1. 设备分配和调度
    2. 设备无关性动作
    3. 设备传输控制
    4. 设备驱动
  • 文件管理
    1. 存储空间管理
    2. 文件操作
    3. 目录操作
    4. 文件和目录存取权限管理

操作系统的特征

  1. 并发
  2. 异步
  3. 共享
  4. 虚拟

操作系统的发展历史

思路：CPU利用率越来越高，资源利用率越来越高

  1. 手工操作（无操作系统）
  2. 单道批处理系统
  3. 多道批处理系统
  4. 分时操作系统

目前使用最广泛的是分时操作系统

分时操作系统的出现背景

  1. 硬件的进步
    • 中断技术的出现
    • 通道的的出现
  2. 任务的需求
    • 多任务的要求
    • 多终端（多用户）的要求

操作系统的扩展

  • 微机操作系统（BIOS将操作系统硬件相对分离，操作系统可以使用BIOS的指令来对硬件进行操作）

  • 实时操作系统

    • 硬实时：非常重视实时的操作，比如导弹控制系统
    • 软实时：相对没有那么重视实时，比如互动网络游戏系统

  • 嵌入式操作系统

    • 软硬件一体，软硬件可以裁剪

    常见的嵌入式操作系统：

    • Andriod
    • Linux
    • WinCE

  • 网络操作系统

    • 普通操作系统+网络通信+网络服务
    • UNIX/Windows/Linux
