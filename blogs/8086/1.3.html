1.3 转移地址的寻址方式

转移地址的寻址方式

方式：段内转移——只改变IP和段间转移——改变了段

段内直接寻址（也叫段内相对寻址）：

...
...
LAST: ...
...
...
JMP LAST
;无条件转移
;同样也有可能是下面这样
...
...
JMP LAST
...
...
LAST: ...
...

8086的条件指令，只能在段内进行转移（范围为-128~127）；无条件指令既可以在段内转移，也可以在段间进行转移。

如果不得不使用段间转移，必须要二级跳（用JMP）来搭桥。

例题：

提示：在实现条件跳转的时候，会有一条指令的生成，该指令的第一个字节为操作码，第二个字节和第三个字节为跳转的量。（如果是段间转移，指令为3字节；如果是段内转移，指令为2字节）

因此这道题的答案为：1000 + 2 + FD(补码) = 1000 + 2 + (-3) = 0FFFH

段内间接寻址

...
MOV BX,OFFSET L1
JMP BX
...
...
L1:...
...

可以搭配用上面存储器的寻址方式5种来寻址。
