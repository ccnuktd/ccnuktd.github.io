1.7 移位，标志寄存器设置和循环类指令

移位类指令

1. 逻辑移位

2. 算数移位

逻辑移位

逻辑左移SHL DST, CNT（将DST中的值逻辑左移位CNT次，DST不能是段寄存器）

如果CNT=1可以直接写SHL AL,1，如果不止一次SHL AL,CL

逻辑右移SHR DST, CNT语法一样

逻辑移位相当于*2和÷2

算术移位

算数左移SAL和逻辑左移一样

算数右移SAR，左边补的是符号位

例题：计算5 * 12(利用移位指令可以加速) 41视频

循环移位

ROL 不带CF的循环左移位 ROR 不带CF的循环右移位

RCL 带CF的循环左移位

RCR 带CF的循环右移位

例题1：若BX = 10110110H，CF = 1，CL = 3，求执行RCR BX, CL后，(BX) = ?

注意这道题有陷阱，BX是16位的，但是题中只给了8位

例题2：（DX : AX） * 16 扩大16倍

对2个字左移1位（*2）可以这样

SHL AX,1

RCL DX,1

标志位操作指令

对CF，DF，IF置位、清零或取反操作 P74页

循环控制指令

  1. LOOP 标号 循环次数已知的情况下可以使用LOOP

    MOV CX,5
L1: ...
    ...
    ...
LOOP L1
;上述程序可以执行4次，如果CX = 0，则循环2^16 - 1 = 65535次

  2. LOOPZ（LOOPE）

    • CX <-- (CX) - 1 （不影响标志）
    • if(CX != 0 && ZF == 1) LOOP循环
    • else 不循环

例题：如何求STRING '1111112111'中不同的位的位置 43视频

小Tip：可以通过$ - STRING得到STRING长度

注意上述说法与length的区别，length是看DUP了多少个length就是多少，而不是字符长度

  1. LOOPNZ（LOOPNE）

把前面的ZF = 1换成ZF = 0，其他一样

用法就是找一个字符

  1. JCXZ 测cx

    if(cx == 0) JMP

    一般用于可能一次都不执行的循环
