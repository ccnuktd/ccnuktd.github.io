4 类型和声明

类型和声明

TIP

1.本类型

布尔量、字符和整数类型放在一起叫做整型。整型和浮点类型合称算术类型。枚举和类被称为用户定义类型。

2.文字量

1.整数文字量：

在整数后面添加，例如：3U指的就是unsigned int；3L指的就是long int

2.浮点文字量

在浮点数后面后面添加，例如：3e8F指的就是float；1e-4L指代是long double

3.大小

4.enum枚举类型

1.一些性质：

1.枚举量的范围：无负数[0 ~ 2^k - 1]，有负数[-2 ^ k ~ 2 ^ k - 1]

2.定义时的语法 名字 + (值)

4.定义与声明的区别

定义：需要建立存储空间。

int add(int a,int b){ return a + b; }

int a = 10;

声明：不需要建立存储空间。

int add(int a,int b);

既声明又定义。

int a = 1;

关于声明和定义的更多知识：https://zhuanlan.zhihu.com/p/162578969
